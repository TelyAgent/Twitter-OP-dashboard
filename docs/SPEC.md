## 1. 数据复盘 `dashboard.html`

### 1.1 认证

- Magic Link 邮箱免密登录
- 登录/登出
- 会话保持

### 1.2 产品组管理

- 多产品组（团队）切换，标签栏展示
- 新增产品组
- 重命名产品组（双击编辑）
- 删除产品组
- 全组总览：聚合所有产品组的概览仪表板

### 1.3 周次管理

- ISO 周选择器
- 切换周次加载对应历史数据
- 填写人记录

### 1.4 NSM 北极星指标

- 指标名称、当前值、上周值、目标值、单位 可编辑
- 环形进度图：当前值 vs 目标值达标率
- 历史趋势折线图：最多 26 周历史数据，叠加目标参考线

### 1.5 漏斗指标

- 漏斗阶段定义（默认：流量 → 互动 → 转化 → 变现）
- 阶段卡片/条形两种可视化视图
- 阶段增删、重命名
- 每个阶段下关联多个指标
- 指标：名称、单位、本周值、上周值、目标值、API 路径 可编辑
- 指标健康分类：ok（达标）/ warn（接近）/ bad（偏离）
- 本周 vs 上周分组柱状图

### 1.6 本周任务

- 任务描述、负责人、状态（未开始/进行中/已完成/阻塞）、关联指标
- 增删改

### 1.7 上周任务复盘

- 同步上周任务到当前周复盘区
- 任务结果标记（已完成/部分完成/进行中/阻塞/已取消）
- 交叉比对：任务关联指标的上周→本周变化（delta%）
- 自动判定：假设成立 / 反向 / 阻塞 / 不显著
- 手动增删复盘任务

### 1.8 复盘总结

- 有效动作（good）
- 无效动作（bad）
- 下周调整（next）

### 1.9 外部 API 集成

- 配置 API 端点：URL、Method、Headers、ResponsePath、POST Body
- 支持 `{team}` `{week}` 占位符
- 一键拉取数据填充到 NSM 和所有指标

### 1.10 全组总览

- 高亮卡片：本周领涨 / 需关注 / 达标团队数
- 总览统计：团队数、达标数、平均完成度、环比增长数
- 团队卡片：NSM 值、delta%、进度条、迷你折线图、跳转入口
- 完成度柱状图：所有团队横向对比

### 1.11 数据持久化与同步

- localStorage 离线优先
- Supabase 异步同步：teams / team_schemas / team_api_configs / weekly_data
- 自动保存（800ms 去抖）+ 手动保存
- 云同步状态指示

### 1.12 UI 偏好

- 主题切换：default / mono / darkhero
- 漏斗视图切换：阶段卡片 / 条形

## 2. 热点雷达 `radar.html`

### 2.1 热点池

- 热点列表，按综合评分降序
- 每个热点：标题、分类标签、评分条、时间/账号数/HOT 标记
- 点击选中查看详情

### 2.2 摘要指标卡

- 总互动量
- 24h 新鲜热点数
- 池总量 + HOT 信号数

### 2.3 热点详情面板

- 摘要文本
- 关键事实
- 机会评估
- 反对观点
- 事件时间线
- 操作按钮：加入观察 / 忽略

### 2.4 关联推文

- 选中热点的所有关联推文，3 列网格
- 按浏览量降序
- 每条推文：作者、时间、正文、互动数据（RT/点赞/回复/浏览）
- 链接到 x.com 原文

### 2.5 评分引擎

- 三维评分：拟合度 / 传播力 / 新鲜度
- HOT 信号：top 15% + 绝对阈值（10k 浏览 or 200 互动）

### 2.6 AI 情报生成

- 选中热点后异步调用 AI Pipeline
- 生成摘要、事实、机会、时间线
- 结果回写数据库缓存

## 3. 模板库 `templates.html`

### 3.1 模板矩阵

- 4 分类 × 7 角度网格总览
- 颜色编码：fire（>=50k 均浏览）/ solid（20-50k）/ weak（<=20k）/ 空
- 图例说明
- 总模板数、fire 数统计

### 3.2 精选模板

- 展示使用次数最高的热门模板
- 模板骨架 + 槽位说明
- 历史使用记录列表
- 命中率、均浏览、总浏览统计
- 版本信息、元数据

### 3.3 最近模板

- 按角度分组展示
- 每个模板：分类、状态、骨架文本、槽位、使用按钮、删除按钮

### 3.4 爆款推文池

- 从热点池加载 hot_signal 推文
- 按角度标签页筛选
- 每条推文：作者、正文、时间、互动数据、角度徽章

### 3.5 AI 提炼模板

- 从爆款推文池选 Top 5 → AI 提炼骨架
- 从爆款推文手工选推文 → AI 提炼骨架
- 提炼结果可编辑（分类、角度、骨架）
- 勾选保存到模板库

### 3.6 AI 填充模板

- 选择模板 → 粘贴素材文本 → AI 生成填充结果
- 结果可编辑、可复制

### 3.7 记录模板使用

- 选择模板 + 粘贴推文 URL
- 抓取推文互动指标
- 保存使用记录（模板 ID、推文 URL、指标）

### 3.8 删除模板

- 带确认弹出框的删除

## 4. 监控源 `sources.html`

### 4.1 来源管理

- 来源表格：账号/类型/7天热点命中/7天趋势迷你图/爆款贡献/状态/添加人
- 行内编辑 Handle
- 状态变更：主力 / 观察 / 退役
- 删除来源（带确认弹出框）
- 类型标签筛选（Twitter KOL / 链上钱包 / 媒体 RSS / AI 推荐）
- 列表分页：每页 15 条，页码导航（首页/上一页/数字/下一页/末页），总条数+页码显示
- 排序：按添加时间倒序（最新在前），相同时间按 handle 字母序稳定排序
- 切换筛选或数据重载后自动回到第 1 页

### 4.2 指标卡片

- 账号总数
- 本周触发热点次数
- 爆款贡献源数
- 沉默账号数（>=4 周无活动）

### 4.3 批量导入

- X List 导入：粘贴 List URL → 拉取成员列表
- 手动文本区导入：每行一个 handle/URL/地址
- 自动类型检测（twitter / onchain / rss / market）
- 预览面板：handle、类型、简介、重复状态
- 导入后自动 PM 相关度评估

### 4.4 单源同步

- 拉取最近 7 天推文（每次最多 100 条，按互动质量截断取 top 30）
- 聚类（4h 窗口）
- 构建热点候选 → 评分 → 入库 hotspots
- 回写来源指标（hits、fire、7 天趋势）

### 4.5 PM 相关度评估

- 拉取来源推文 → 计算预测市场相关度分数
- 星级展示（★★★ / ★★☆ 等）
- 筛选：仅 PM 相关（★★+）

### 4.6 批量同步

- 同步全部 PM 相关来源（pm_score >= 0.4）
- 限流安全：24h 同源去重、8-25s 随机源间延迟、单次上限 20 源
- 429/403 全局 10 分钟退避，剩余源自动跳过
- 启动前确认弹窗（排队数量 + 预计耗时）

### 4.7 爆款推文信息流

- 展示所有 hot_signal=true 的推文
- 按浏览量降序
- 含图片/视频媒体

### 4.8 手动添加爆款推文

- 粘贴推文 URL → 抓取数据
- 预览：作者、时间、角度、正文、互动指标
- 自动分类 → 保存到 hotspots

### 4.9 AI 推荐

- 展示 AI 推荐新增的监控源
- 操作按钮：加入主力 / 观察 / 忽略

### 4.10 评审管理

- 评审日期横幅
- 自动汇总摘要
- 手动评审入口
- 表格摘要：显示数/总数、本周新增、退役、待 review
